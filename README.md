# Gamepedia

Gamepedia — настольное приложение на Python (Tkinter), служащее интерактивной энциклопедией видеоигр.  
Все данные хранятся в MySQL, поддерживаются регистрация, роли и административная панель для управления каталогом.

## Возможности

- Регистрация и авторизация пользователей (роли `admin` и `user`)
- Админ-панель с добавлением, редактированием и удалением игр
- Отправка предложений об играх в файл `proposals.txt`
- Поиск и фильтрация по названию, жанру, платформе, году и рейтингу
- Отображение баннеров из каталога `banners/`
- Встроенный блокнот для заметок

## Роли

- **Администратор** — видит «Админ-панель», управляет записями и очищает дубликаты.  
  При первом запуске автоматически создаётся пользователь `admin / admin123`.
- **Пользователь** — просматривает каталог и отправляет предложения в `proposals.txt`.

## Структура проекта

- `main.py` — GUI, фильтрация, формы входа и админ-панель
- `db.py` — функции доступа к MySQL (отредактируйте параметры подключения под себя)
- `banners/` — изображения игр (`<название>.jpg`)
- `proposals.txt` — накопитель пользовательских предложений
- `gamepedia.sql` — пример структуры таблиц и наполнения БД

## Настройка БД

1. Создайте базу `gamepedia` в MySQL.
2. Подключите `gamepedia.sql`, чтобы создались таблицы `games` и `users` и заполнились их  данными.
3. Проверьте настройки подключения в `db.py` (хост, логин, пароль).

## Запуск

1. Установите Python 3 (включая Tkinter).
2. Установите зависимости:
   ```bash
   pip install mysql-connector-python pillow ttkbootstrap
   ```
3. Запустите приложение:
   ```bash
   python main.py
   ```
