# Gamepedia

Gamepedia — настольное приложение на Python (Tkinter + ttkbootstrap), служащее интерактивной энциклопедией видеоигр. Все данные (игры, пользователи, избранное, предложения) хранятся в MySQL, поэтому изменения мгновенно доступны всем пользователям.

## Возможности

- Регистрация и авторизация с ролями `admin` и `user`. Администратор `admin/admin123` создаётся автоматически при первом запуске.
- Гибкий каталог: поиск, фильтры по букве, жанру, платформе, году и сортировка по рейтингу. При выборе игры показывается подробное описание и баннер `banners/<Название>.jpg`.
- Встроенная система избранного. Пользователь может добавлять игры в свою подборку и открывать отдельное окно с личным списком.
- Отправка предложений на добавление игр. Заявки попадают в таблицу `proposals`, где администратор просматривает детали, переносит игру в основной список или удаляет предложение.
- Админ-панель: добавление, редактирование, удаление игр, очистка дубликатов и работа с предложениями в пару кликов.
- Мини-блокнот/скетчбук для заметок, который сохраняет рисунки в папку `notes`.

## Структура проекта

- `main.py` — графический интерфейс, фильтры, аутентификация, избранное и админ-панель.
- `db.py` — функции доступа к MySQL (при необходимости измените `host`, `user`, `password`, `database`).
- `database/gamepedia.sql` — схема и тестовые данные для таблиц `users`, `games`, `proposals`, `favorites`.
- `banners/` — изображения игр (по названию).

## Требования

- Python 3.10+ с установленным Tkinter.
- MySQL 8+ или совместимый сервер БД.
- Зависимости Python:

  ```bash
  pip install mysql-connector-python pillow ttkbootstrap
  ```

## Настройка базы

1. Создайте базу `gamepedia` в MySQL.
2. Импортируйте `database/gamepedia.sql`, чтобы получить таблицы и пример данных.
3. Подправьте настройки подключения в `db.py`, если используете другую БД/учётку.

## Запуск

```bash
python main.py
```

После старта появится окно авторизации.

- **Обычный пользователь** может просматривать каталог, предлагать новые игры и работать с избранным.
- **Администратор** получает кнопку «Админ-панель» с полным управлением каталогом и предложениями.

## Советы

- Чтобы у игры был баннер, положите файл `banners/<Название>.jpg`.
- Избранное и предложения жёстко связаны с пользователями, поэтому не забывайте входить под нужным аккаунтом.
- Изменения через админ-панель сразу пишутся в MySQL: список игр пересоздаётся и синхронизируется с БД.
